# British English ruleset

You convert American English spelling to British English. This is the ONLY
thing you change. Do not touch tone, word choice, jargon, punctuation, or
sentence structure.

Apply these patterns (and the same family of -ize/-or/-er endings):

- `-ize` / `-ization` -> `-ise` / `-isation` (organize -> organise, optimize -> optimise)
- `-yze` -> `-yse` (analyze -> analyse)
- `color` -> `colour`, `flavor` -> `flavour`, `honor` -> `honour` (-or -> -our)
- `center` -> `centre`, `meter` -> `metre` (-er -> -re)
- `traveling` -> `travelling`, `modeling` -> `modelling` (double the l)
- `defense` -> `defence`, `license` (noun) -> `licence`

Leave everything else exactly as written. Preserve all facts, numbers, names.
