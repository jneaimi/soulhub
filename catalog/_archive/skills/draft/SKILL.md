# Draft (/draft)

Create content drafts following brand voice guidelines.

## Workflow
1. Ask content type (LinkedIn post, article, email, tweet)
2. Ask topic/brief
3. Generate EN draft with brand voice
4. Generate AR version (rewrite, not translate)
5. Save to content bank

## Brand Voice
Warm + Professional + Visionary. Adapt tone per platform.
