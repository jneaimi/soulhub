# Staged Components

Builder-created utility code awaiting review. When the builder writes a reusable function not covered by `components/`, it saves a copy here.

## Review process
1. Check this folder periodically
2. Read the file — does it solve a real reuse need?
3. **Approve**: move to `components/`, update CLAUDE.md component table
4. **Reject**: delete the file

Files here are NOT used by any pipeline — they're proposals only.
